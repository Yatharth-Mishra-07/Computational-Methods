Computational methods I learn in my coursework implemented using python

**For bisection method:**

enter the function f(x). All math module functions and x allowed. Example: x^3 - x^2, sin(x) - tan(x), etc.

enter the start and end of the intervals in which the program will look for roots.

enter accuracy ( (actual root - approximate root) will be less than the given accuracy. write 1e-n for result accurate up to n decimal points)



**For secant method:**

enter the function f(x). All math module functions and x allowed. Example: x^3 - x^2, sin(x) - tan(x), etc.

enter the the initial guess values (x0 and x1)

enter the desired accuracy 


**For Newton Raphson method:**

enter the function f(x). All math module functions and x allowed. Example: x^3 - x^2, sin(x) - tan(x), etc.

Enter the derivative f'(x) of the given function f(x). If the wrong derivative is given, the answer will be wrong as well.

Enter a guess value
